我的第一個網頁練習

從空白檔案開始，記錄第一次完成首頁結構、樣式與手機版調整的過程。

03DESIGNING THE WEB

第一次建立完整網頁時，我先從最基本的頁首、主內容與頁尾開始。畫面雖然簡單，但每一個區塊都讓我更理解 HTML 結構和 CSS 樣式如何合作。

先完成可以閱讀的結構

我先確認標題、段落、導覽與按鈕的順序，再處理顏色、間距和字體。這樣遇到問題時，比較容易知道是內容結構還是樣式設定造成的。

再把桌面版帶到手機

完成桌面版後，我用較窄的寬度檢查標題、卡片與按鈕。手機版不是只把內容縮小，而是重新安排閱讀順序與操作空間。

第一個作品不需要完美，但要完整到能讓自己看見下一步。
